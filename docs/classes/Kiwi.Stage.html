Kiwi.Stage Class

Defined in: src/core/Stage.ts:9
Module: Kiwi

Each game contains a single Stage which controls the creation and management of main domElements required for a Kiwi game to work. Such as the Canvas and the rendering contexts, as well as the width/height of the game and the position it should be on the screen.
