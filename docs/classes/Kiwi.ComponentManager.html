Kiwi.ComponentManager Class

Defined in: src/core/ComponentManager.ts:9
Module: Kiwi

The component manager is a class that is used to handle components that are active on a particular object. Any object that has a component manager attached to it can use components. If you want to check to see if a particular component is on an object you can ask the component manager, or when updating components you can tell the component manager to update and all of the components will update as well.
