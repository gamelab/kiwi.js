onKeyDown

Signal public

Defined in src\input\Keyboard.ts:111

A Signal that dispatches events when a key is pressed/is down. This mimics the natural 'keydown' event listener, so it will keep dispatching events if the user holds the key down. Note: This fires after the 'onKeyDownOnce' signal.

onKeyDownOnce

Signal public

Defined in src\input\Keyboard.ts:122

A Signal that dispatches events when a key is pressed/is down initially. This event only fires the first time that the key is pressed, so it won't dispatch events if the user is holding the key down. Note: This fires before the 'onKeyDown' signal;
